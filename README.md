# NemoClaw-agritwin-dairy-agentic-ai-platform-backend

Backend services for AgriTwin Dairy Agentic AI Platform

## Modules
- api-gateway: Cow, Herd, Sensor
- ml-inference: MilkYield, HealthAlert
- agent-orchestrator: AgentRecommendation

## Routes
- animals
- health
- milk-yield
- alerts
- recommendations

## Shared Dependency
All routes import entities from `@robo-jedi-pl/agritwin-shared`.
